5 - Code N chill

Este proyecto se trata de una aplicación desarrollada con el mero objetivo de darle al usuario, una herramienta para combinar sonidos relajantes de distintos tipos de ambientes, y así, este pueda crear un espacio de trabajo relajante y placentero, facilitando la productividad y estimulando la concentración.

La inspiración para hacerlo vino a mi cabeza cuando me di cuenta de lo cansado que estaba de depender de una busqueda en youtube para crear mi ambiente de trabajo ideal, por lo que decidí desarrollar una aplicación a la que siempre pueda recurrir, eximiendome de esta necesidad de entrar a la mencionada plataforma de videos.
